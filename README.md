# Personal Hub
A scala play framework project which will attempt to include a variety of different pieces of functionality.

## Quick start

### Prerequisites
- [sbt](http://www.scala-sbt.org/)
- MongoDB

### Running the application

To run the application on the default port, run:
```
sbt run
```

To run this on a specific port, run:
```
sbt "run <port>"
```

### Testing the application

To run the tests, run:
```
sbt test
```

To run coverage, run:
```
sbt clean coverage test coverageReport
```
